# fun_with_lld
A repo for learning Low level system design and design patterns.
The aim of this repo is to understand the popular design patterns through diagrams and code.
The repo will compromise of code in python and golang in the py and go modules respectively.
Each pattern will be accompanied by at least one use case in the pratice folder.

## Singleton Pattern Use Cases

1. Design a logging system.
2. Design a Database connection pool management.
